# MRF Geolocation from Amsterdam Inference Output

This module takes the `Ams_All.csv` file generated by the Amsterdam panorama inference script and estimates final object geolocations.

## Input

The expected input file is:

```text
Ams_All.csv
```

with the following columns:

```text
Fname, Pix, ObjBearing, Lon, Lat, Depth, RFov, LFov, Captured_at
```

If the `Depth` column is empty, the script uses a default depth value of `7.5 m`.

## Method

The script performs the following steps:

1. Reads the detected positive image crops and their metadata from `Ams_All.csv`.
2. Converts each detection into a camera ray using camera latitude, longitude, and object/view bearing.
3. Computes pairwise ray intersections between detections.
4. Filters invalid intersections using camera-distance and object-distance constraints.
5. Applies MRF optimization to select geometrically consistent pairwise intersections.
6. Clusters selected intersections spatially.
7. Exports final object geolocations and the image/view pairs supporting each final location.

## Example usage

```bash
python run_mrf_geolocation.py \
    --input-csv /path/to/Ams_All.csv \
    --output-dir /path/to/mrf_outputs
```

## Outputs

```text
final_geolocations.csv
```

Final estimated object locations:

```text
cluster_id, lat, lon, score
```

where `score` is the number of selected pairwise intersections supporting that cluster.

```text
final_geolocation_pairs.csv
```

Image/view pairs supporting each final geolocation:

```text
cluster_id, det_i, det_j, fname_i, fname_j, intersect_lat, intersect_lon, ...
```

This file can be used to inspect which pair of detections/images contributed to each final object location.

```text
admissible_intersections.csv
```

All geometrically valid intersections before MRF optimization.

```text
selected_intersections.csv
```

Intersections selected by the MRF before final clustering.
